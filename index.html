About

「AIって便利そうだけど、
自分には難しそう」
そう思っていませんか？

Claude Code は、Anthropic が開発した対話型のコーディング AI です。でも、使い方はエンジニアだけのものではありません。

この勉強会では、コードを書いたことがない方でも、Claude Code を「専属コンシェルジュ」として育てながら、自分の業務に使えるツールを実際に作る体験ができます。

「見て学ぶ」ではなく「手を動かして学ぶ」ハンズオン形式。当日その場で、動くものを完成させましょう。

What you'll learn

この勉強会で学べること

3 つのテーマを軸に、Claude Code を仕事で使いこなすための基礎から応用まで体験します。

psychology
01 / 基礎理解

Claude Code とは何か

AI との「会話の仕方」を学びます。どんな指示を出せば思い通りに動くか、コンシェルジュとして育てるコツを体験しながら身につけます。

terminal
02 / 実践体験

実際に手を動かして作る

勉強会の中で自分専用の小さなツールを実際に作成します。「動いた！」という感覚を体験することで、活用イメージが一気に広がります。

rocket_launch
03 / 業務応用

自分の仕事への活かし方

作ったものをどう日常業務に組み込むか。自動化・効率化・情報整理など、自分のシーンに合わせた活用アイデアを持ち帰れます。
